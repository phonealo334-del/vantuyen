## Mô Tả
TinPhat ERP là một hệ thống quản lý doanh nghiệp toàn diện được thiết kế để giúp các công ty quản lý:
- Nhân sự
- Sản phẩm
- Kho hàng
- Khách hàng
- Tài chính
- Hợp đồng
- Báo cáo

## Tính Năng Chính

✅ Quản lý Nhân Sự
✅ Quản lý Sản Phẩm
✅ Quản lý Kho Hàng
✅ Quản lý Khách Hàng
✅ Quản lý Tài Chính

✅ Cài Đặt Hệ Thống

## Cấu Trúc Thư Mục
```
TinPhat-ERP/
├── login.html                    # Trang đăng nhập
├── index.html                    # Trang chính
├── assets/
│   ├── css/                     # Các tệp CSS
│   ├── js/                      # Các tệp JavaScript
│   ├── images/                  # Hình ảnh
│   └── vendor/                  # Thư viện bên thứ ba
└── pages/                       # Các trang nội dung
```

## Bắt Đầu
1. Clone repository
2. Mở `login.html` để đăng nhập
3. Sau đó truy cập `index.html` để sử dụng ứng dụng
